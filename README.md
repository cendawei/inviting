### inviting邀请函
- css3+vue实现前端动画交互
- 单页面（spa）应用模式
- 支持地图定位
- 支持留言和签到等功能
- 小图标集成雪碧图
- 一键发布
### 技术栈
> express+sequelize+MySQL+ejs+gulp+webpack+vue+vue-router+sprites+babel+es6
### 项目安装
> `npm install`
### 前端项目构建
> `npm run dev`
### 运行项目
> `npm run start`
### 项目部署
> 服务启动：`npm run server` <br />
> 前端项目构建：`npm run build`
